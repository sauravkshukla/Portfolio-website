# Resume File

To add your resume, please follow these steps:

1. Create your resume as a PDF file
2. Save it as `Saurav_Kumar_Shukla_Resume.pdf` in this directory
3. The "View Resume" button will download this file when clicked

## Note
The resume file needs to be added manually as it's a binary file. You can:
- Create a PDF using tools like Google Docs, Word, Canva, or any PDF editor
- Name it exactly as: `Saurav_Kumar_Shukla_Resume.pdf`
- Place it in the `/public/resume/` directory

This will be automatically downloadable from the portfolio website.
